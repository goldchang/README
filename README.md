##Tips for Beginners
If you run into problems related to the **Terminal** application, **don't panic.**<br>
It's because you don't have the required dependencies.<br>
Simply run<br>
**`./vendor/cm/get-prebuilts`**<br>
And It'll fetch the APK & Shared Libraries.
